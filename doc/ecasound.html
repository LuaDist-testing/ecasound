NAME

ecasound.lua - does whatever

SYNOPSIS

 local E = require 'ecasound'
 a = { 6,8,7,9,8 }
 b = { 4,7,5,4,5,6,4 }
 local probability_of_hypothesis_being_wrong = E.ttest(a,b,'b>a')

DESCRIPTION

This module does whatever

FUNCTIONS

ttest(a,b, hypothesis)

The arguments a and b are arrays of numbers

The hypothesis can be one of 'a>b', 'a<b', 'b>a', 'b<a', 'a~=b' or 'a<b'.

ttest returns the probability of your hypothesis being wrong.
